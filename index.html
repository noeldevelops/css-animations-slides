Why is CSS so easy? 🥰

All the CSS ever written (sorta): 

selector {
    property: value;
}
a {
    color: aquamarine;
}

CSS has a simple syntax & the properties tend to be human-readable - they're in English too...

Here are some of the common properties MDN CSS Properties Reference

Yet it's so powerful! CSS Zen Garden is a great example of the possibilities...so is the entire web!

Why is CSS so hard? 😈

An elegant system, far more powerful than any framework, designed for the realities of a flexible, multilingual, multi-device web.
(From "The way we talk about CSS" by Rachel Andrew)

1. Everything is interrelated.

Cascading, specificity, property sets, oh my!

2. It's fail-proof!

"There are a heaps of these [common] 'bugs'. They aren’t actually bugs, merely a developers lack of understanding as to what the browser is doing. More correctly; what your CSS code is telling the browser to do." Ben Frain

3. Browsers are different and the language is constantly evolving

In order to write good CSS, you have to know which browsers support which features, and when to use hacks or fallbacks. This requires both knowledge and experience. Can I Use? is a good starting point.

How can you write better CSS, more easily? 💅

If you take CSS seriously you can do some amazing things with it, which is very satisfying because the syntax is super easy to learn.

  1. Leverage browser brain power 🧠
  2. Think globally 🌍
  3. Stay organized 📋
  4. Use tools 🧰

Leverage the browser's brain power 🧠

"If you find yourself wrestling with CSS layout, it’s likely you’re making decisions for browsers they should be making themselves."
(From Every Layout) 

   🚫 Don't write this in your CSS! 🚫
    width: 500px;

Think globally 🌍

Remember the cascade? It's there to make you more efficient! It is much easier to write styles that apply to many elements first, then add more specific overrides as you need them.

For example, the link tag. Start with a generic element, then get more specific:


    a {
    color: aquamarine;
    }
    
    a.button {
    border: 2px;
    padding: 1em;
    color: red;
    }

Stay Organized 📋


    /**********************/ 
    /* Navigation Styles */
    /**********************/
    nav {
        display: flex;
    .....
    /**********************/ 
    /* Button Styles */
    /**********************/
    a.button {
        margin: 0 auto;
        padding: 0.5em;
        font-size: 20px;
        font-family: "Newsreader", serif;
    ....
